# Security Policy

## Reporting a Vulnerability
If you discover a security issue in this project, please do not report it publicly at first.

Instead, please open a private report through GitHub or contact the project maintainer directly.

Please include:
- A clear description of the issue
- Steps to reproduce it
- The affected source file(s)
- The possible impact

## Supported Versions
This is a student project, so only the latest version is currently supported.
